# Instructions And Layout
## 📝 Registers
| Register | Purpose |
|----------|---------|
| A | Accumulator |
| B | Base Register |
| C | Counter |
| D | Data Register |
| E | Extra Register |
| F | Flags Register |

## 🔧 Instructions
| Instruction | Description |
|-------------|-------------|
| `mov` | Move data between registers |
| `add` | Add values |
| `push` | Push to stack |
| `pop` | Pop from stack |
| `halt` | Stop execution |

## 📚 Stack
```
+----------+
|    ...   | <- Stack grows downward
+----------+
```
